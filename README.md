## 🚀 Como executar

1. Instale o **Java JDK**
2. Clone o repositório:
   ```bash
   git clone https://github.com/Raphael2005x/beecrowd-solutions.git
   ```
3. Abra a pasta no seu editor preferido
4. Compile e execute os arquivos `.java` individualmente
